# Run and deploy on AI Studio 

View in AI Studio: https://ai.studio/apps/drive/1zYKpFF4gu890vsJw_pthG_ouraTnDoHS

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
